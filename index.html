Overview

What is Standard View?

시장에 대한 이해부터 기업 분석까지, 워크플로우를 단일 인텔리전스 플랫폼으로 통합합니다.

Standard View는 (1) 회계감사 업무에 필수적인 산업 및 기업에 대한 이해와 (2) 개인 투자에서 필요한 재무, 산업, 매크로, 딜 정보를 한 곳에서 수집·분석·기록하기 위해 설계된 개인용 통합 인텔리전스 대시보드입니다.

DART OpenAPI 기반 재무제표 조회부터 AI가 요약하는 산업/매크로 브리프, 매일 텔레그램으로 수신하는 Daily Brief, 그리고 SQLite에 축적되는 인사이트 로그까지 — 투자 검토의 전 과정을 지원합니다. 로컬 앱에서 깊은 분석을 수행하고, Telegram으로 매일 시장 동향을 파악하며, GitHub Pages로 날짜별 리포트를 아카이브합니다.

Audit Use Case

회계감사 업무 활용

피감기업과 산업을 빠르게 이해하고, 감사계획 단계의 위험평가와 분석적 절차를 보조하는 리서치 도구로 활용합니다.

Standard View는 감사 업무에서 DART 재무정보, 산업 뉴스, 매크로 환경, 딜성 이슈, 인사이트 로그를 결합해 감사인이 확인해야 할 질문과 리스크 포인트를 구조화하는 데 활용할 수 있습니다. 감사증거를 대체하거나 감사의견을 자동으로 형성하는 도구가 아니라, 감사인의 전문가적 판단을 보조하는 리서치·분석 플랫폼으로 설계되었습니다.

🏢

피감기업 이해

재무제표 기반으로 피감기업의 성과와 이상 징후를 빠르게 파악합니다.

  • DART 재무제표 기반 성장성·수익성·안정성 파악
  • 3개년 추세와 주요 재무비율을 통한 이상 징후 확인
  • 동종 기업 비교를 통한 상대적 위치 파악
🌐

산업 및 외부환경 이해

산업·매크로 환경이 피감기업에 미치는 영향을 분석합니다.

  • 산업 뉴스와 글로벌 트렌드 기반 업황 변화 확인
  • 금리·환율·물가·경기 등 매크로 환경의 피감기업 영향 분석
  • 업종별 주요 리스크와 회계 이슈 도출
⚠

위험평가 보조

재무·외부 리스크를 식별해 감사계획 단계의 위험평가를 보조합니다.

  • 매출 급증·수익성 악화·부채비율 상승·현금흐름 악화 등 재무 리스크 식별
  • 산업 침체·규제 변화·자금조달 환경 악화 등 외부 리스크 정리
  • 감사계획 단계에서 유의적으로 검토할 영역 후보 도출
📊

분석적 절차 보조

주요 재무 지표의 추세와 비정상적 변동 후보를 식별합니다.

  • 매출·영업이익·순이익·자산·부채·자본의 연도별 추세 확인
  • 영업이익률·ROE·부채비율 등 주요 비율 변동 분석
  • 동종 기업·관련 산업 흐름과 비교해 비정상적 변동 후보 식별
💬

감사 질문 리스트 생성

CFO 인터뷰와 후속 감사절차 설계에 활용할 질문 후보를 정리합니다.

  • CFO·회계팀 인터뷰 확인 질문 정리
  • 매출 인식·손상·충당부채·금융부채·특수관계자 거래 관련 질문 후보
  • 후속 감사절차 설계에 참고할 체크포인트 제공
💡

인사이트 로그 활용

감사 중 발견한 이슈와 분석 결과를 로컬 DB에 저장·관리합니다.

  • 주요 이슈·뉴스·분석 결과를 인사이트 로그에 저장
  • 기업·산업·출처·감사상 시사점·후속 확인사항 기록
  • 감사계획서·내부 메모·리뷰 대응 시 참고자료로 활용
⚖

한계 및 주의사항

  • ·Standard View의 분석 결과는 감사증거가 아니라 감사인의 판단을 보조하는 참고자료입니다.
  • ·외부 뉴스와 AI 요약은 원천자료 확인이 필요합니다.
  • ·최종 감사절차와 결론은 감사인이 독립적으로 판단해야 합니다.
Comparison

Local App vs Telegram Brief

두 채널은 각각 다른 목적으로 설계되어 상호 보완적으로 동작합니다.

항목 ⚙ Local App ✈ Telegram Brief 📋 Audit Use
목적 깊이 있는 심층 분석 작업 공간 매일 받아보는 투자·딜 모닝 브리프 피감기업 재무분석, 산업 이해, 위험평가, 분석적 절차 보조
실행 방식 사용자가 직접 기업·산업·키워드 입력 정해진 시간 완전 자동 실행 감사 대상 기업명 또는 업종 입력 후 수동 실행
결과 형태 대시보드, 인터랙티브 차트, 상세 분석 짧은 요약 메시지 + GitHub HTML 링크 재무 추세 차트, 뉴스 브리프, 인사이트 로그
사용 상황 특정 기업·딜을 집중 분석할 때 매일 시장 동향을 빠르게 파악할 때 감사계획 수립, 위험평가, 현장 준비 단계
장점 맞춤형 깊이 있는 인사이트 별도 실행 없이 자동으로 정보 수신 반복 리서치 시간을 줄이고 판단·질문 설계에 집중
데이터 기록 SQLite 인사이트 로그, Deal Memo GitHub Pages에 날짜별 HTML 아카이브 인사이트 로그에 감사상 시사점·후속 확인사항 축적
Why it matters

왜 Standard View인가

투자 심사에 필요한 정보의 수집·분석·기록 사이클을 최소한의 마찰로 실행합니다.

⚡

실시간 대기 시간 최소화

사전 생성·캐시·발송 구조로 매일 아침 별도 작업 없이 최신 인텔리전스를 수신합니다.

🎯

역할 분리로 집중력 확보

로컬 앱은 심층 분석, Telegram은 일별 시장 파악으로 목적에 맞게 채널을 분리합니다.

📚

날짜별 리포트 아카이브

GitHub Pages에 날짜별 HTML 리포트가 누적되어 시장 변화와 과거 분석을 재검토할 수 있습니다.

🗃

개인 리서치 데이터베이스 구축

인사이트 로그로 뉴스·메모·분석을 SQLite에 축적해 나만의 투자 리서치 DB를 만들어갑니다.

🔍

감사에서도 외부 환경을 함께 봐야 한다

피감기업의 재무정보뿐 아니라 산업, 매크로, 자금조달, 딜 이슈를 함께 이해해야 실질적인 위험을 파악할 수 있습니다.

🗂

감사계획·위험평가 단계의 리서치 속도 향상

여러 정보를 하나의 화면에 모아 감사계획과 위험평가 단계에서 빠르게 참고할 수 있습니다.

🎯

반복 리서치를 줄여 판단에 집중

반복적인 리서치 시간을 줄이고, 감사인이 더 중요한 판단과 질문 설계에 집중할 수 있게 합니다.
